## 414.第三大的数

### 冒泡排序

```java
    public int thirdMax(int[] nums) {
        // 1、降序排序
        for(int i = 0; i < nums.length; i++){
            for(int j = 0; j < nums.length - i - 1; j++){
                if(nums[j] < nums[j+1]){
                    int temp = nums[j];
                    nums[j] = nums[j+1];
                    nums[j+1] = temp;
                }
            }
        }

        // 2、找去重后第三大的数
        int rank = 3;
        for(int i = 0; i < nums.length; i++){
            if(i+1 < nums.length && nums[i] == nums[i+1]){
                continue;
            }

            if(--rank == 0){
                return nums[i];
            }
        }

        return nums[0];
    }
```



### TreeSet

```java
    public int thirdMax(int[] nums) {
        TreeSet<Integer> set = new TreeSet<>();
        for(int i = 0; i < nums.length; i++){
            set.add(nums[i]);
            if(set.size() > 3){
                set.pollFirst();
            }
        }

        return set.size() >= 3 ? set.first() : set.last();
    }
```
