## 189.轮转数组

### 使用额外数组

```java
    public void rotate(int[] nums, int k) {
        int len = nums.length;
        // 长度可能超出数组长度，所以先求余
        k = k % len;
        int[] rt = new int[len];
        int j = 0;
        // 将待轮转数据放到临时数组开头
        for(int i = len-k; i < len; i++){
            rt[j++] = nums[i];
        }

        // 顺移剩下元素
        for(int i = 0; i < len-k; i++){
            rt[j++] = nums[i];
        }

        // 替换原数组
        for(int i = 0; i < len; i++){
            nums[i] = rt[i];
        }
    }
```
