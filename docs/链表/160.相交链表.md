# [160] 相交链表



```java
	public ListNode getIntersectionNode(ListNode headA, ListNode headB) {
        ListNode p1 = headA, p2 = headB;
        boolean b1 = true, b2 = true;

        // 思想是，A + B 等于 B + A
        // p1按照A B顺序前进，p2按照B A顺序前进，第一个相等的节点就是交点
        while(p1 != p2){
            p1 = p1.next;
            if(p1 == null && b1){
                p1 = headB;
                b1 = false;
            }

            p2 = p2.next;
            if(p2 == null && b2){
                p2 = headA;
                b2 = false;
            }
        }

        return p1;
    }
```



这种没有添加标志位，但是没有交点也不会重复切换，因为A+B和B+A如果没有交点则在末尾null处因为p1等于p2会退出循环（巧妙）

```java
    public ListNode getIntersectionNode(ListNode headA, ListNode headB) {
        ListNode p1 = headA, p2 = headB;

        // 思想是，A + B 等于 B + A
        // p1按照A B顺序前进，p2按照B A顺序前进，第一个相等的节点就是交点
        while(p1 != p2){
            if(p1 == null){
                p1 = headB;
            }else{
                p1 = p1.next;
            }

            if(p2 == null){
                p2 = headA;
            }else{
                p2 = p2.next;
            }
        }

        return p1;
    }
```
