# [141] 环形链表



```java
    public boolean hasCycle(ListNode head) {
        ListNode slow = head, fast = head;

        // 成环总会相遇
        // 不成环，则fast指针先到达末尾
        while(fast != null && fast.next != null){
            fast = fast.next.next;
            slow = slow.next;

            if(fast == slow){
                return true;
            }
        }

        return false;
    }
```



```java
    public ListNode middleNode(ListNode head) {
        ListNode slow = head, fast = head;

        while(fast != null && fast.next != null){
            fast = fast.next.next;
            slow = slow.next;
        }

        return slow;
    }
```
