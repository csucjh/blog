# [876] 链表的中间结点



```java
    public ListNode middleNode(ListNode head) {
        ListNode slow = head, fast = head;

        while(true){
            if(fast.next == null){
                break;
            }

            if(fast.next.next == null){
                slow = slow.next;
                break;
            }

            fast = fast.next.next;
            slow = slow.next;
        }

        return slow;
    }
```



```java
    public ListNode middleNode(ListNode head) {
        ListNode slow = head, fast = head;

        while(fast != null && fast.next != null){
            fast = fast.next.next;
            slow = slow.next;
        }

        return slow;
    }
```
