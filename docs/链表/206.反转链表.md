# [206] 反转链表



```java
    public ListNode reverseList(ListNode head) {
        if(head != null){
            ListNode p1 = head, p2 = head.next;

            ListNode p; p1.next = null;
            while(p2 != null){
                p1 = p2.next;
                p2.next = head;
                // 互换p1 p2
                p = p1; p1 = p2; p2 = p;
                head = p1;
            }
        }

        return head;
    }
```



其实反转链表就是一个头插法，（jdk7里面hashmap不就是因为这个原因会有死循环，所以jdk8改成了尾插法）

```java
    public ListNode reverseList(ListNode head) {
        if(head == null){
            return head;
        }
        
        ListNode prev = null, curr = head;    
        ListNode temp;
        while(curr != null){
            temp = curr.next;
            curr.next = prev;
            prev = curr;
            curr = temp;
        }

        return prev;
    }
```



**对于递归算法，最重要的就是明确递归函数的定义**。具体来说，我们的 `reverse` 函数定义是这样的：

**输入一个节点 `head`，将「以 `head` 为起点」的链表反转，并返回反转之后的头结点**。

```java
    int count = 0;

    private void print(int count){
        for(int i = 0; i < count; i++){
            System.out.print("————");
        }
    }

    public ListNode reverseList(ListNode head) {
        print(count++);
        System.out.println("head val:" + head.val);
        if(head.next == null){
            print(--count);
            System.out.println("return:" + head.val);
            return head;
        }

        ListNode last = reverseList(head.next);

        head.next.next = head;
        head.next = null;

        print(--count);
        System.out.println("return:" + last.val);
        return last;
    }
```
